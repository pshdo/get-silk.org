Set-ModuleVersion

Updates a module's version.

Syntax

Set-ModuleVersion [-ManifestPath] <String> [[-SolutionPath] <String>] [[-AssemblyInfoPath] <String>] [[-ReleaseNotesPath] <String>] [[-NuspecPath] <String>] [[-Version] <Version>] [[-PreReleaseVersion] <String>] [[-BuildMetadata] <String>] [<CommonParameters>]

Parameters

Name Type Description Required? Pipeline Input Default Value
ManifestPath String

The path to the module's manifest.

 true false
SolutionPath String

The path to the module's manifest.

 false false
AssemblyInfoPath String

Path to an C# file to update with the assembly version.

 false false
ReleaseNotesPath String

Path to a release notes file.

 false false
NuspecPath String

Path to the module's Nuspec file.

 false false
Version Version

The version to build. If not provided, pulled from the module's manifest.

 false false
PreReleaseVersion String

The pre-release version, e.g. alpha.39, rc.1, etc.

 false false
BuildMetadata String

Build metadata.

 false false
