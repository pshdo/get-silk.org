New-ModuleHelpIndex

Creates an index page for a module's help.

Syntax

New-ModuleHelpIndex [-ModuleName] <String> [[-Script] <String[]>] [[-TagsJsonPath] <String>] [<CommonParameters>]

Parameters

Name Type Description Required? Pipeline Input Default Value
ModuleName String

The name of the module whose index page to create.

 true false
Script String[]

The names of any scripts that should be included.

 false false
TagsJsonPath String

The path to the tags file. If not provided, no tag tab is generated.

 false false
